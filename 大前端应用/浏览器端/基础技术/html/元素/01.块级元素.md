# 块级元素 

HTML 中的元素可分为两种类型，块级元素和行内元素，下面让我们来了解一下什么是块级元素吧！

## 块级元素 

块级元素最大的特点就是会自动换行，它通常会以新行来开始，并且以新行来结束（独占一行）。可以通过 CSS 属性来设置块级元素的宽高、行高、外边距和内边距。

块级元素的宽度默认是容器的 100%， 它可以容纳内联元素和其他的块级元素。

```css 
.block {
    display: block;
}
```
