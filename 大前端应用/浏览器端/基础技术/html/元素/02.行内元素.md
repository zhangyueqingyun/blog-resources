# 行内元素

HTML 中的元素可分为两种类型，块级元素和行内元素，下面让我们来了解一下什么是块级元素吧！
内联元素或称为行内元素 一般都是基于语义级的基本元素，只能容纳文本或者其他内联元素。

## 1 行内元素的特点

行内元素和相邻的行内元素会分布在一行中，设置行内元素的宽度和高度是无效的，默认的宽度是它本身的宽度。可以设置行内元素水平方向的 padding 和 margin，而垂直方向上的无效。

行内元素只能容纳其他行内元素。

## 2 行内元素的 CSS 表示

```css 
.block {
    display: inline;
}
```
