# Google 编入索引的文件类型

Google 可将大多数类型的网页和文件内容编入索引。最常编入索引的文件类型包括：

- Adobe 便携式文档格式 (.pdf)
- Adobe PostScript (.ps)
- Google 地球 (.kml.kmz)
- GPS 交换格式 (.gpx)
- Hancom Hanword (.hwp)
- HTML (.htm .html 其他)
- Microsoft Excel (.xls、xlsx)
- MicrosoftPowerPoint (.ppt .pptx)
- Microsoft Word (.doc、.docx)
- OpenOffice 文本文档 （.odt）
- 富文本格式 (.rtf)
- 可缩放矢量图形 (.svg)
- Tex/LateX (.tex)
- 文本文件 (.txt .text 其他文件扩展名) 包括采用常用编程语言的源代码：
    - Basic 源代码 (.bas)
    - C/C++ 源代码 (.c .cc .cpp .cxx .h .hpp)
    - C# 源代码 (.cs)
    - java 源代码 (.java)
    - Perl 源代码 (.pl)
    - Python 源代码 (.py)
- 无线标记语言 (.wml .wap)
- XML (.xml)

您可以在 Google 搜索中使用 filtetype: 运算符将搜索结果限制为特定文件类型或文件扩展名。例如, filetype:rtf galway 会搜索以 .rtf 结尾并且内容包含了 galway 一词的 RTF 文件和网址。
