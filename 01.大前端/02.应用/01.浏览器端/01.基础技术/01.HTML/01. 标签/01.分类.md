# HTML 标签分类

## 1. 基本标签

```
<!DOCTYPE>      // 文档类型
<html>          // html 文档
<title>         // 文档标题
<body>          // 文档主体
<h1> - <h6>     // 1-6 级标题
<p>             // 段落
<br>            // 换行
<hr>            // 水平线
<!-- -->        // 注释
```

## 2. 格式标签

```
<abbr>          // 缩写
<address>       // 文档作者、所属者联系信息
<b>             // 粗体文本
<bdi>           // 设置一段脱离父元素文本方向的文本
<bdo>           // 定义文本方向
<blockquote>    // 块引用
<cite>          // 引用
<code>          // 计算机代码文本
<del>           // 被删除文本
<dfn>           // 定义项
<em>            // 强调文本
<i>             // 斜体文本
<ins>           // 插入文本
<kbd>           // 键盘文本
<mark>          // 带有记号的文本
<meter>         // 度量衡，仅用于已知最大和最小值的度量
<pre>           // 预格式文本
<progress>      // 运行中的任务进度（进程）
<q>             // 短的引用
<rp>            // 不支持 ruby 元素的浏览器所显示的内容
<rt>            // 字符的解释或发音
<ruby>          // ruby 注释（中文注音或字符）
<s>             // 加删除线的文本
<samp>          // 计算机代码样本
<small>         // 小号文本
<strong>        // 强调文本
<sub>           // 下标文本
<sup>           // 上标文本
<time>          // 日期/时间
<u>             // 下划线文本
<var>           // 文本的变量部分
<wbr>           // 在文本中的何处添加换行符
```

## 3. 表单

```
<form>          // 表单
<input>         // 输入
<textarea>      // 多行文本输入
<button>        // 按钮
<select>        // 列表
<optgroup>      // 选择列表中相关选项的组合
<option>        // 选择列表中的选项
<label>         // input 元素的标注
<fieldset>      // 围绕表单中元素的边框
<legend>        // fieldset 元素的标题
<datalist>      // input 元素可能的选项列表
<keygen>        // 用于表单的密钥对生成器字段
<output>        // 计算结果
```

## 4. 框架

```
<iframe>        // 内联框架
```

## 5. 图像

```
<img>           // 图像
<map>           // 图像映射
<area>          // 图像地图内部区域
<canvas>        // canvas 图像
<figcaption>    // caption for a <figure> element
<figure>        // 组合元素
```

## 6. 影音视频

```
<audio>         // 音频
<source>        // 媒体资源
<track>         // 外部文本轨道
<video>         // 音频或视频
```

## 7. 链接

```
<a>             // 链接
<link>          // 文档与外部资源的关系
<main>          // 文档的主体关系
<nav>           // 导航链接
```

## 8. 列表

```
<ul>            // 无序列表
<ol>            // 有序列表
<li>            // 列表项
<dl>            // 定义列表
<dt>            // 定义定义列表中的项目
<dd>            // 定义列表中项目的描述
<menu>          // 菜单列表
<command>       // 用户可能调用的命令（单选按钮、复选框或者按钮）
```

## 9. 表格

```
<table>         // 表格
<caption>       // 表格标题
<th>            // 表格中的表头单元格
<tr>            // 表格中的行
<td>            // 表格中的单元
<thead>         // 表格中的表头内容
<tbody>         // 表格中的主体内容
<tfoot>         // 表格中的脚注
<tfoot>         // 表格中一个或多个列的属性值
<col>           // 一个或多个列的属性值
<colgroup>      // 表格中共格式化的列组
```

## 10. 样式/节

```
<style>         // 文档的样式信息
<div>           // 文档中的节
<span>          // 文档中的节
<header>        // 文档头部
<footer>        // 文档底部
<section>       // 文档的某个区域
<article>       // 文章内容
<aside>         // 所处内容之外的内容
<details>       // 用户课件或隐藏的需求的补充细节
<dialog>        // 对话框或者窗口
<summary>       // 定义可见的标题，当用户点击标题时会显示出像是信息
```

## 11. 元信息

```
<meta>          // html 文档的元信息
<base>          // 页面中所有链接的默认地址或默认目标
```

## 12. 程序

```
<script>        // 客户端脚本 
<noscript>      // 针对不支持客户端脚本的用户的替代内容
<embed>         // 容器，用于嵌入外部应用或者互动程序（插件）
<object>        // 嵌入的对象
<param>         // 对象的参数
```
