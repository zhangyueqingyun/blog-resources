# HTML 标签分类

## 1. 基本标签

```
<!DOCTYPE>      // 文档类型
<html>          // html 文档
<title>         // 文档标题
<body>          // 文档主体
<h1> - <h6>     // 1-6 级标题
<p>             // 段落
<br>            // 换行
<hr>            // 水平线
<!-- -->        // 注释
```

## 2. 格式标签

```
<abbr>          // 缩写
<address>       // 文档作者、所属者联系信息
<b>             // 粗体文本
<bdi>           // 设置一段脱离父元素文本方向的文本
<bdo>           // 定义文本方向
<blockquote>    // 块引用
<cite>          // 引用
<code>          // 计算机代码文本
<del>           // 被删除文本
<dfn>           // 定义项
<em>            // 强调文本
<i>             // 斜体文本
<ins>           // 插入文本
<kbd>           // 键盘文本
<mark>          // 带有记号的文本
<meter>         // 度量衡，仅用于已知最大和最小值的度量
<pre>           // 预格式文本
<progress>      // 运行中的任务进度（进程）
<q>             // 短的引用
<rp>            // 不支持 ruby 元素的浏览器所显示的内容
<rt>            // 字符的解释或发音
<ruby>          // ruby 注释（中文注音或字符）
<s>             // 加删除线的文本
<samp>          // 计算机代码样本
<small>         // 小号文本
<strong>        // 强调文本
<sub>           // 下标文本
<sup>           // 上标文本
<time>          // 日期/时间
<u>             // 下划线文本
<var>           // 文本的变量部分
<wbr>           // 在文本中的何处添加换行符
```
